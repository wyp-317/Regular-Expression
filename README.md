# 📦 快递收件信息正则解析器

本项目为编译原理小组的正则表达式应用实现。项目使用 Python 内置的 `re` 模块，通过灵活构造正则表达式，实现了对快递文本中**姓名**、**手机号**以及**省、市、区、详细地址**的智能分离与提取。

## 🌟 核心亮点 (正则特性展示)
1. **命名捕获组 (Named Capturing Groups)**：使用 `(?P<name>...)` 精准映射 `province` / `city` / `district` 到字典中。
2. **零宽断言 (Zero-width Assertions)**：
   - 手机号提取使用了**负向先后视断言** `(?<!\d)` 和 `(?!\d)`，防止匹配到超长订单号中的11位数字。
   - 姓名提取兜底策略中使用了**正向先行断言** `(?=...)`，成功在未分割的字符串中精准定位位于省市之前的姓名。
3. **乱序容错**：通过结合字符串的 Chunking 与正则表达式筛查特征（排除匹配 `(?:省|市|区|县|大厦)$` 等），有效支持了 `地址+手机号+姓名` 等各种乱序的用户输入。
4. **直辖市/自治区适配**：支持“北京市”、“内蒙古自治区”、“上海浦东新区”（无市字）等特殊地理名词的准确分割。

## 🛠️ 运行环境要求
- **Python 版本**：Python 3.6 或以上版本。
- **依赖库**：纯原生的 Python `re` 标准库，无需额外 `pip install` 安装其他第三方库。
